The files in the resources directory will be available when running Python code cells as if they are in the current working directory. For example, the code below lists the files in the current working directory, which are the same as those in the resources directory. import os os.listdir() We can perform normal file operations such as reading a text file, or writing a new file: with open("data.txt") as f: content = f.read() print(content) with open("new_file.txt", "w") as f: f.write("This is a new file.") import os os.listdir() # This should show the new file We can also use packages as normal to open files. For example, here we use pandas to read a CSV file: import pandas as pd data = pd.read_csv("ages.csv") data.head() This could be useful for loading datasets to analyse in examples or exercises.
